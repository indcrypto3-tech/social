# Autopostr

Autopostr is the all-in-one platform for creators and agencies to plan, analyze, and automate their social media presence.

## Features

- **AI-Powered Scheduling**: Automatically schedule your posts for the best times.
- **Analytics**: Deep dive into your social media performance.
- **Content Calendar**: Visualize your content strategy.
- **Multi-Platform Support**: Connect X (Twitter), LinkedIn, Instagram, and more.

## Getting Started

1.  Clone the repository.
2.  Install dependencies: `npm install`
3.  Set up environment variables in `.env`.
4.  Run the development server: `npm run dev`
